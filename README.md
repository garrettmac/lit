## Installation

- Install `react-native` first

```bash
npm i react-native -g
yarn add 
mkdir Example
cd Example
yarn install #(or with npm)
npm start
```


### Example

or clone the repo and play with the example project

```bash
$ git clone https://github.com/garrettmac/
$ cd /Example
$ yarn install
$ npm start
```
